# Installation

If you need wsl run:
```powershell
wsl --instal -d Ubuntu-20.04
```

Activate wsl with:
```powershell
wsl
```

If it's your first time, you'll need to set up your user profile. 

Then run the shell located here with:
```bash
bash ./wsl_setup.sh
```

Follow the prompts during runtime to execute the download.
